# AmirnetApp_React - הכנה למבחן אמירנט

A high-performance React application designed to optimize English vocabulary acquisition for the Amirnet/Psychometric exams through immersive media consumption.

🚀 Key Features
Synchronized Logging: Real-time event tracking to timestamp and log vocabulary during media playback.

Offline First: Localized English-Hebrew dictionary implementation for zero-latency, offline translations.

Academic Modules: Curated academic databases partitioned into 50-word high-intensity study units.

Dynamic State Filtering: Logic-based categorization (Known/Unknown/Uncertain) for targeted revision.

LLM Integration: Leveraging Claude AI for contextual disambiguation and automated assessment generation.

🛠️ Tech Stack
Core: React.js (Vite)

UI/UX: Tailwind CSS (Engineered with full RTL support)

Architecture: Functional Components, Hooks, Context API

DevOps: Git, GitHub, Vercel (CI/CD)

📈 Objective
To bridge the gap between passive entertainment and active learning, providing a data-driven tool for Israeli students to master academic English.
